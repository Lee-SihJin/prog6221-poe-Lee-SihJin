# 🛡️ Cybersecurity Awareness Chatbot

A command-line based cybersecurity awareness chatbot that helps users learn about online safety practices through interactive conversations. 


## ✨ Features

### Core Features
- **Voice Greeting** - Plays a recorded welcome message on startup
- **ASCII Art Logo** - Visual cybersecurity-themed header
- **Interactive Chat** - Natural conversation about cybersecurity topics
- **Security Topics** - Covers passwords, phishing, and safe browsing
- **Input Validation** - Handles empty inputs and invalid queries gracefully
- **Enhanced UI** - Colored text, borders, typing effects, and emojis
- **Personalized Experience** - Asks for and uses user's name throughout

### Security Topics Covered
- **Password Safety** - Creating strong passwords, password managers, 2FA
- **Phishing Protection** - Identifying phishing emails, suspicious links
- **Safe Browsing** - HTTPS, public Wi-Fi safety, browser security
